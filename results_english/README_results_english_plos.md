# English-language PLOS ONE result files

This folder contains English-language structured result CSV files prepared for PLOS ONE data availability review.

All column names, categorical values, province names, locality labels, family-node labels, model terms, diagnostic labels, and table labels are provided in English or Romanized English.

Chinese reference columns from the original historical source have been removed from this English-language version. The original Chinese structured files are retained separately in the results/ folder.

These English-language result files are sufficient to reproduce and verify the tables, robustness checks, diagnostic outputs, negative-control analysis, and node-removal simulation reported in the manuscript.
